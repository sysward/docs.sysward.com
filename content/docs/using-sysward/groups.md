<a href="/images/groups.png" target=_blank><img src="/images/groups.png" alt="SysWard Groups page listing groups and their agent counts" class="img-fluid rounded"/></a>

Groups let you organize your servers so you can view, patch, and report on
related machines together (for example by environment, role, or team). The
**Groups** page lists every group with the number of agents in it.

<a href="/images/groups_new_group.png" target=_blank><img src="/images/groups_new_group.png" alt="Creating a new group" class="img-fluid rounded"/></a>

Create a group with **New Group** and give it a name. Assign servers to it from
the [Agents](/docs/using-sysward/agents/) list or the
[dashboard](/docs/using-sysward/dashboard/) using the inline group dropdown, and
remove a group with **Delete**.

<a href="/images/group_overview.png" target=_blank><img src="/images/group_overview.png" alt="Group overview showing all agents in a group" class="img-fluid rounded"/></a>

Select a group name to open its overview: every agent in the group, where you
can apply updates across the whole group at once and review each server's
history.
