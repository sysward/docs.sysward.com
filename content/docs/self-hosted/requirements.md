## System requirements

* Operating System: Ubuntu 20 LTS, 22 LTS, or 24 LTS
* Internet connectivity to reach the license server

### Minimum

* CPU: 1 Cores
* Memory: 4GB RAM
* Disk: 70GB

### Recommended

* CPU: 2 Cores
* Memory: 8GB RAM
* Disk: 80GB
