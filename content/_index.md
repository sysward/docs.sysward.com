SysWard keeps your Linux fleet patched and secure. Install a lightweight agent
on each server and SysWard reports pending updates, matches installed packages
against known CVEs, and lets you apply fixes across your infrastructure from a
single dashboard.

## Where to start

- **[Getting Started](/docs/getting-started/quick-start/)** — install the agent and get a server reporting in under a minute.
- **[Using SysWard](/docs/using-sysward/dashboard/)** — the dashboard, agents, groups, packages, CVEs, and remediation.
- **[Account & Billing](/docs/account-billing/organization/)** — organization settings, users, your profile, and subscription.
- **[Integrations](/docs/integrations/webhooks/)** — Slack, Discord, and email notifications.
- **[Self-Hosted Appliance](/docs/self-hosted/requirements/)** — run SysWard on your own infrastructure.
- **[Reference](/docs/reference/supported-operating-systems/)** — supported operating systems, agent commands, and release notes.
